# MyTube - Frontend (Client)

## 📌 Hướng dẫn chạy thử giao diện

1. **Di chuyển vào thư mục client:**
   ```bash
   cd client
   ```
2. **Cài đặt các thư viện:**
   ```bash
   npm install
   ```
3. **Khởi chạy ứng dụng:**
   ```bash
   npm run dev
   ```
4. **Trải nghiệm:**
   * Mở trình duyệt và truy cập: **("http://localhost:3000')"**
---

## ============================================
##      ⚠️ Lưu ý quan trọng khi chấm bài
## ============================================

> [!WARNING]
> **Giới hạn trải nghiệm khi chạy độc lập Frontend:**
>
> * Đây là mã nguồn phần **Frontend** (giao diện người dùng).
> * Do **không có máy chủ Backend và Cơ sở dữ liệu (Database)** chạy kèm, các tính năng yêu cầu đăng nhập hoặc lưu trữ dữ liệu (như: *Đăng nhập, Đăng ký, Lịch sử xem, Danh sách đăng ký, Quản lý Studio, Video đã thích, Tải video lên...*) sẽ **không hoạt động hoàn chỉnh** hoặc không hiển thị dữ liệu động.
> * Để sử dụng được toàn bộ các tính năng và trang web, hệ thống cần được kết nối đồng thời với Backend và Cơ sở dữ liệu.
